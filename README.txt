RE:제로 전시회 수정본

사용법
1. ZIP 파일 압축을 풉니다.
2. index.html을 더블클릭합니다.
3. GitHub Pages에 올릴 때는 index.html과 images 폴더를 함께 업로드합니다.

이번 구성
- Dress To Impress / Dream1128: 4개 전시실로 축소, 총 44장
- Dress To Impress / 밍태: 1개 전시실, 총 6장
- Dress To Impress / 뵵주: 1개 전시실, 총 4장
- Dress To Impress / 2인 합작 특별관: 1개 전시실, 총 2장, Designed by Dream1128 × 밍태
- Style Stars / Dream1128: 1개 전시실, 총 8장
- Style Stars / 밍태: 1개 전시실, 총 5장
- Style Stars / 뵵주: 1개 전시실, 총 2장

수정 방법
- 작품명과 작품 해설은 index.html을 메모장으로 열고 customPieceText 구역에서 수정합니다.
- 랜덤 판결문은 index.html을 메모장으로 열고 DEFAULT_ORACLE_LINES 구역에서 추가/수정합니다.
- 전시장 안에는 작품명/해설 수정 버튼과 판결문 수정 버튼이 없습니다.

주의
- creator, image, id는 건드리지 않는 것을 추천합니다.
- 가치 평가 서명서는 작품당 한 번만 제출할 수 있고, 제출 후에는 수정/삭제 버튼으로 관리합니다.
- HALL 버튼을 누르면 원하는 전시실로 바로 이동할 수 있습니다.
